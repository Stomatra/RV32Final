ID 阶段：
  识别 funct7 == 7'b0000001
  根据 funct3 判断是哪条 M 指令

ID/EX：
  保存 m_op
  保存 rs1/rs2
  保存 rd
  标记这是一条 M 指令

EX 阶段：
  如果是 MUL 类，计算乘法结果
  如果是 DIV/REM 类，启动多周期除法器

流水线控制：
  除法器 busy 时 stall PC、IF/ID、ID/EX、EX/MEM
  除法器 done 时把结果送入写回路径

WB：
  rd <= m_result

  | 指令       | funct3 | 含义               |
| -------- | -----: | ---------------- |
| `MUL`    |  `000` | 乘法低 32 位         |
| `MULH`   |  `001` | 有符号 × 有符号，高 32 位 |
| `MULHSU` |  `010` | 有符号 × 无符号，高 32 位 |
| `MULHU`  |  `011` | 无符号 × 无符号，高 32 位 |
| `DIV`    |  `100` | 有符号除法            |
| `DIVU`   |  `101` | 无符号除法            |
| `REM`    |  `110` | 有符号取余            |
| `REMU`   |  `111` | 无符号取余            |
